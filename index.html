Mercari 100円値下げ App

このリポジトリは Vercel 上でホスティングできるように構成されています。Vercel 上では Python API エンドポイントが動作しますが、Mercari のブラウザ自動化はローカル実行が必要です。

API Status Trigger Check
注意:
  • Mercari の値下げ自動化は Selenium と Chrome を必要とします。
  • Vercel のサーバーレス環境ではブラウザ自動化はサポートされません。
  • このページは Vercel デプロイの確認用です。

Use git clone https://github.com/henmiryotaro-droid/merukariapp.git to get the repository and run locally.

If you want to run the Mercari automation itself, deploy the project to a server or local environment that supports Chrome and Selenium.
